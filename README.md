# Youth Mentor Board

Youth Mentor Board 是一个面向青年辅导、课程跟进与反馈记录管理的中型协作平台，提供仪表盘、列表页、详情页、创建页和接口层，适合在容器环境中直接启动，也适合作为继续扩展业务流程、报表和通知能力的基础工程。

## 适用场景

- 团队围绕固定业务对象管理状态、优先级、时间线和协作备注。
- 管理者通过首页汇总查看当前规模、待处理事项和重点指标。
- 业务成员通过页面和接口共同维护同一套数据现场。

## 核心能力

- 首页仪表盘展示总览指标、提醒信息和近期动态。
- 列表页支持浏览核心记录，并提供检索或筛选能力。
- 详情页展示结构化字段、阶段状态和说明信息。
- 创建页支持录入新记录，并带有基础表单校验。
- 提供 JSON API，方便后续接入移动端、自动化脚本或报表服务。

## 技术栈

- Python、Flask、Jinja2、原生前端页面

## 目录结构

- 页面与接口代码放在 repo 目录内的主语言工程结构中。
- static、public、templates、views 等目录用于承载前端页面和静态资源。
- data 目录用于预留运行期数据文件。
- patch 目录用于后续补丁文件留存。

## Docker 启动

docker build -t repo .
docker run --rm -p 8080:8080 -p 2222:22 repo

## 本地启动

pip install -r requirements.txt && python -m app.server
